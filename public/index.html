Group Processes

Our group worked well together for the first group assessment however there were some issues in the way we approached some things which we are working towards improving for this assessment. One of the key issues we had is that tasks would often be completed far past agreed upon deadlines. To improve on this, we have assigned a project manager who will be able to work to ensure all tasks are completed by their agreed upon dates.

Another area that we faced some challenges was communication, which we will now focus on improving through greater use of the team chat functionality, while ensuring if there is a certain group member we are directing a comment to we will tag them in the comment, so it is not missed.

The last major pain point is some tasks were a bit heavier than we thought they were and assigned a lot to one or two people without realising. This assessment we are ensuring we have determined correct sizes of each section and divided accordingly to make sure all group members have an evenly distributed section.
